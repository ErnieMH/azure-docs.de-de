Die VMware-Softwareversionen in neuen Bereitstellungen von Private Cloud-Clustern von Azure VMware Solution sind wie folgt:

| Software              |    Version   |
| :---                  |     :---:    |
| VCSA/vSphere/ESXi |    6.7 U3l    | 
| ESXi                  |    6.7 U3l    | 
| vSAN                  |    6.7 U3l    |
| NSX-T <br />**HINWEIS:** NSX-T ist die einzige unterstützte Version von NSX.               |      2.5.2     |


Die aktuell ausgeführte Softwareversion wird auf neue Cluster, die einer vorhandenen privaten Cloud hinzugefügt werden, angewendet. Weitere Informationen finden Sie in den [Anforderungen der VMware-Softwareversion](https://docs.vmware.com/en/VMware-HCX/services/user-guide/GUID-54E5293B-8707-4D29-BFE8-EE63539CC49B.html).
