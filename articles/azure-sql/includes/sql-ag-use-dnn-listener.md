> [!NOTE]
> Kunden mit SQL Server 2019 CU8 und höher unter Windows 2016 und höher können den herkömmlichen VNN-Listener und Azure Load Balancer durch einen [DNN-Listener (Distributed Network Name, Name des verteilten Netzwerks)](../virtual-machines/windows/availability-group-distributed-network-name-dnn-listener-configure.md) ersetzen. Überspringen Sie die restlichen Schritte dieses Artikels, in denen der Listener und das Lastenausgleichsmodul erstellt werden.
