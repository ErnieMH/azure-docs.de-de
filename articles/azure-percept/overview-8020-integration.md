# <a name="azure-percept-dk-8020-integration-overview"></a>Azure Percept DK 80/20-Integrationsübersicht

Das Azure Percept DK und Audiozubehör wurden für die Integration in das [80/20-Schienensystem](https://8020.net/) entwickelt.

## <a name="8020-features"></a>80/20-Features

Jede Hardwarekomponente wird mit den Aussparungen und Ausbuchtungen versehen, um in den 1010-Extrusionstyp zu passen. Dank dieser Integration können Kunden und Lösungsersteller ihre Proof of Concepts leichter auf Produktionsumgebungen ausweiten.
