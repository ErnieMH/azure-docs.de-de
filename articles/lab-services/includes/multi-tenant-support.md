Wenn Sie Administrator oder Lab-Besitzer sind, der Teil mehrerer Azure AD-Mandanten ist, können Sie zwischen Mandanten über das Azure Lab Services-Portal (in der oberen rechten Ecke) navigieren. 

![Auswählen eines Mandanten](../media/multi-tenant-support/picker.png)

Nachdem Sie einen Mandanten ausgewählt haben, wählen Sie in der linken oberen Ecke ein Lab-Konto innerhalb Ihres Azure AD-Mandanten (Azure Active Directory) aus.
