# <a name="azure-policy-built-in-definitions-for-azure-networking-services"></a>Integrierte Azure Policy-Definitionen für Azure-Netzwerkdienste

Diese Seite ist ein Verzeichnis der integrierten [Azure Policy](../governance/policy/overview.md)-Richtliniendefinitionen für Azure-Netzwerkdienste. Weitere Azure Policy-Integrationen für andere Dienste finden Sie unter [Integrierte Azure Policy-Richtliniendefinitionen](../governance/policy/samples/built-in-policies.md).

Die Namen der einzelnen integrierten Richtliniendefinitionen sind Links zur entsprechenden Richtliniendefinition im Azure-Portal. Verwenden Sie den Link in der Spalte **Version**, um die Quelle im [Azure Policy-GitHub-Repository](https://github.com/Azure/azure-policy) anzuzeigen.
