# <a name="security-control-penetration-tests-and-red-team-exercises"></a>Sicherheitskontrolle: Penetrationstests und Red Team-Übungen

Testen Sie die Gesamtstärke der Verteidigung einer Organisation (die Technologie, die Prozesse und die Personen), indem Sie die Ziele und Aktionen eines Angreifers simulieren.

## <a name="111-conduct-regular-penetration-testing-of-your-azure-resources-and-ensure-remediation-of-all-critical-security-findings"></a>11.1: Durchführen regelmäßiger Penetrationstests Ihrer Azure-Ressourcen und Sicherstellen der Behebung aller kritischen Sicherheitsergebnissen

|--|--|--|
| 11,1 | 20.1, 20.2, 20.3, 20.4, 20.5, 20.6, 20.7, 20.8 | Shared |

Nutzen Sie die Microsoft-Strategie und Durchführung von Red Team- und Livewebsite-Penetrationstests für von Microsoft verwaltete Cloudinfrastruktur, Dienste und Anwendungen.
