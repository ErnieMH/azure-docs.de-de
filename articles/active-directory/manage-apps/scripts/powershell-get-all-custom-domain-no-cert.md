# <a name="get-all-azure-ad-proxy-application-apps-published-with-no-certificate-uploaded"></a>Abrufen aller Apps für den Azure AD-Anwendungsproxy, die ohne ein hochgeladenes Zertifikat veröffentlicht wurden
