Zum Bezeichnen von Äußerungen stehen zwei Möglichkeiten zur Verfügung: Entitätenpalettenbezeichnung und Inlinebezeichnung. 

**Entitätenpalettenbezeichnung**

Wenn Sie oben rechts auf das Symbol **@** klicken, wird die Entitätenpalette rechts auf der Seite geöffnet.

1. Wählen Sie die Entität aus, die als Erstes bezeichnet werden soll.

    :::image type="content" source="../media/how-to-label-entity-palette.png" alt-text="Bezeichnen einer Entität mithilfe der Entitätenpalette" lightbox="../media/how-to-label-entity-palette.png":::

1. Markieren Sie den Text, den Sie mit dieser Entität bezeichnen möchten.
    
    :::image type="content" source="../media/how-to-label-entity-palette-2.png" alt-text="Bezeichnen einer Entität mithilfe der Entitätenpalette" lightbox="../media/how-to-label-entity-palette-2.png":::

**Inlinebezeichnung**

1. Markieren Sie den Text, den Sie bezeichnen möchten.
2. Wählen Sie im daraufhin angezeigten Menü die Entität aus, mit der Sie den Text bezeichnen möchten.

    :::image type="content" source="../media/how-to-label-inline.png" alt-text="Menü zum Bezeichnen des Texts" lightbox="../media/how-to-label-inline.png":::
