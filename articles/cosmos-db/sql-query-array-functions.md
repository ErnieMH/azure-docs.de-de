# <a name="array-functions-azure-cosmos-db"></a>Arrayfunktionen (Azure Cosmos DB)

Die Array Funktionen ermöglichen das Ausführen von Vorgängen für Arrays in Azure Cosmos DB.

## <a name="functions"></a>Functions

Die folgenden Skalarfunktionen führen einen Vorgang für einen Arrayeingabewert durch und geben einen numerischen Wert, booleschen Wert oder Arraywert zurück:

* [ARRAY_CONCAT](sql-query-array-concat.md)
* [ARRAY_CONTAINS](sql-query-array-contains.md)
* [ARRAY_LENGTH](sql-query-array-length.md)
* [ARRAY_SLICE](sql-query-array-slice.md)
