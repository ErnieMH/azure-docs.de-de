# <a name="getting-started-with-azure-iot-embedded-device-development"></a>Einstieg in die Entwicklung eingebetteter Azure IoT-Geräte

**Gilt für**: [Entwicklung von eingebetteten Geräten](about-iot-develop.md#embedded-device-development)

Dieser Leitfaden für die ersten Schritte enthält eine Reihe von Schnellstarts, die Ihnen zeigen, wie Sie mit eingebetteten Geräten und Azure IoT arbeiten. 

In jeder Schnellstartanleitung führen Sie die folgenden grundlegenden Aufgaben aus:
* Installieren eines Satzes von Tools zur Entwicklung eingebetteter Geräte zum Programmieren eines bestimmten Geräts in C
* Erstellen eines Images, das Azure RTOS-Komponenten und-Beispiele enthält, und anschließendes Ausführen eines Flashes für das Gerät
* Sicheres Herstellen einer Verbindung eines Geräts mit Azure IoT
* Anzeigen der Gerätetelemetriedaten, Anzeigen von Eigenschaften und Aufrufen von Cloud-zu-Gerät-Methoden
