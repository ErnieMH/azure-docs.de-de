# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Anzeigen des Bestands Ihrer Instanzen im Azure-Portal

Nachdem Sie Ihre [Metriken, Protokolle](upload-metrics-and-logs-to-azure-monitor.md) oder [Nutzungsdaten](view-billing-data-in-azure.md) hochgeladen haben, können Sie Ihre Instanz als Azure-Ressource im Azure-Portal anzeigen. Um Ihre Ressource im Azure-Portal anzuzeigen, navigieren Sie zur speziellen URL [https://portal.azure.com](https://portal.azure.com), und führen Sie die folgenden Schritte aus:

1. Navigieren Sie zu **Alle Dienste**.
1. Suchen Sie nach Ihrem Datenbankinstanztyp.
1. Fügen Sie den Typ Ihren Favoriten hinzu.
1. Wählen Sie im linken Menü den Instanztyp aus.
1. Zeigen Sie Ihre Instanzen in derselben Ansicht wie Ihre anderen Azure SQL- oder Azure PostgreSQL Hyperscale-Ressourcen an (verwenden Sie Filter, um eine präzisere Ansicht zu erhalten).
