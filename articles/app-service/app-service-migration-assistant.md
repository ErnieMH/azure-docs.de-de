# <a name="migrate-to-azure-app-service"></a>Migrieren zu Azure App Service

Mithilfe [App Service Migration Assistant](https://azure.microsoft.com/services/app-service/migration-assistant/) können Sie Ihre lokale App auf Azure App Service migrieren. App Service Migration Assistant wurde entwickelt, um Ihre Reise in die Cloud durch eine kostenlose, einfache und schnelle Lösung für die Migration von Anwendungen von On-Premises in die Cloud zu vereinfachen.

Mit dem Azure App Service Migration Assistant können Sie schnell:

- Ihre App-URL scannen, um zu beurteilen, ob sie ein guter Kandidat für die Migration ist
- Laden Sie den Migration Assistant herunter, um die Migration zu starten.
- Verwenden Sie das Tool, um Bereitschaftstests und eine allgemeine Bewertung der Konfigurationseinstellungen Ihrer App durchzuführen
- Migrieren Sie Ihre App oder Site über das Tool zu Azure App Service.

Erfahren [Sie, wie Sie Web-Apps zu Azure-App-Dienst migrieren](https://www.youtube.com/watch?v=9LBUmkUhmXU).
