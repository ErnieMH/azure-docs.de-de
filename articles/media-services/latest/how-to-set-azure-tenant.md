# <a name="how-to-find-your-azure-tenant-id"></a>Ermitteln der Azure-Mandanten-ID

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

[!INCLUDE [quickstarts-free-trial-note](../../../includes/quickstarts-free-trial-note.md)]

## <a name="find-your-tenant-id-in-the-portal"></a>Suchen Ihrer Mandanten-ID im Portal

1. Melden Sie sich beim [Azure-Portal](https://portal.azure.com) an.
1. Wählen Sie unter Azure-Dienste das Azure Active Directory-Symbol aus. Die Active Directory-Seite wird geöffnet.
1. Ihre Mandanten-ID befindet sich unter **Grundlegende Informationen**.
1. Kopieren Sie die Mandanten-ID, und fügen Sie sie für später in ein beliebiges Textdokument ein.
