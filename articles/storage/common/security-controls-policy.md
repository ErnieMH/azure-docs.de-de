# <a name="azure-policy-regulatory-compliance-controls-for-azure-storage"></a>Azure Policy-Kontrollen zur Einhaltung gesetzlicher Bestimmungen für Azure Storage

[Die Einhaltung gesetzlicher Bestimmungen in Azure Policy](../../governance/policy/concepts/regulatory-compliance.md) bietet von Microsoft erstellte und verwaltete Initiativendefinitionen (als _integriert_ bezeichnet) für die **Compliancedomänen** und **Sicherheitskontrollen**, die mit unterschiedlichen Compliancestandards zusammenhängen. Auf dieser Seite werden die **Compliancedomänen** und **Sicherheitskontrollen** für Azure Storage aufgeführt. Sie können die integrierten Elemente für eine **Sicherheitskontrolle** einzeln zuweisen, um Ihre Azure-Ressourcen mit dem jeweiligen Standard kompatibel zu machen.
