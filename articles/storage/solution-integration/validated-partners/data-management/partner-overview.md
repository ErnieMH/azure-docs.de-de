# <a name="azure-storage-data-governance-management-and-migration-partners"></a>Partner für die Azure Storage-Datengovernance, -verwaltung und -migration

In diesem Artikel werden die Partnerunternehmen von Microsoft vorgestellt, die eine Azure Storage-Integration und Verbesserungen Ihrer allgemeinen Datenverwaltungsfunktionen anbieten. Diese Partnerlösungen unterstützen etwa Speicherbewertung und -berichterstellung, plattformunabhängige Migration, Replikation, Cloudtiering oder Datengovernance.
