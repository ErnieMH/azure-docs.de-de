# <a name="azure-storage-analytics-and-big-data-partners"></a>Partner für Azure Storage Analytics und Big Data

In diesem Artikel werden Microsoft Partner vorgestellt, deren Lösungen in Azure Data Lake Storage integriert sind. Diese Partnerlösungen decken Workloads wie z. B. moderne Data Warehouse-Workloads, erweiterte Analysen und Echtzeitanalysen ab. Diese Partner nutzen den [hierarchischen Namespace](../../../blobs/data-lake-storage-namespace.md) in Azure Storage, um Ihre Lösung zu optimieren und effizient in Azure auszuführen.
