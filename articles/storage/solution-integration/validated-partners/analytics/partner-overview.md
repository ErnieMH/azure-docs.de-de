# <a name="azure-storage-analytics-and-big-data-partners"></a>Partner für Azure Storage Analytics und Big Data

In diesem Artikel werden Microsoft Partner vorgestellt, deren Lösungen in Azure Data Lake Storage integriert sind. Sie bieten Lösungen für Workloads wie moderne Data Warehouses, erweiterte Analysen, Echtzeitanalysen und vieles mehr. Diese Partner nutzen den hierarchischen Namespace in Azure Storage, um ihre Lösung zu optimieren und effizient in Azure auszuführen.
