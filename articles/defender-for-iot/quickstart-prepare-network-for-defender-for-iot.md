# <a name="quickstart-prepare-your-network-for-azure-defender-for-iot-deployment"></a>Schnellstart: Vorbereiten Ihres Netzwerks für die Bereitstellung von Azure Defender für IoT

## <a name="choose-your-hardware"></a>Auswählen Ihrer Hardware
Azure Defender für IoT unterstützt verschiedene zertifizierte Appliances. Hierzu zählen unter anderem folgende:
- Zertifizierte vorkonfigurierte Appliances, auf denen bereits Software installiert ist
- Nicht konfigurierte zertifizierte Appliances, auf die Sie Sensorsoftware herunterladen und darauf installieren können
- Spezifikationen der Appliance für die lokale Verwaltungskonsole. Die lokale Verwaltungskonsole ist nicht als vorkonfigurierte Appliance verfügbar. 

Weitere Informationen finden Sie im [Leitfaden zu den Hardwarespezifikationen](https://aka.ms/AzureDefenderforIoTBareMetalAppliance).


## <a name="install-components"></a>Installieren von Komponenten
Nachdem Sie Ihre Hardwarekomponenten erworben und Ihr Softwarepaket über das Azure Defender für IoT-Portal heruntergeladen haben, können Sie Componentware installieren. Weitere Informationen finden Sie im [Installationshandbuch](https://aka.ms/AzureDefenderforIoTInstallSensorISO).

## <a name="onboard-sensors-to-the-azure-defender-for-iot-portal"></a>Durchführen des Onboardings für Sensoren im Azure Defender für IoT-Portal 
 Dieser Artikel enthält einen Überblick über die Bereitstellungs-, Onboarding- und Aktualisierungsprozesse für Azure Defender für IoT-Sensoren und die lokale Verwaltungskonsole. Dieser Prozess wird gemäß den Anweisungen auf der Willkommensseite von Azure Defender für IoT und auf den nachfolgenden Seiten durchgeführt.

![Screenshot: Azure-Willkommensseite](media/updates/image4.png)

So öffnen Sie die Willkommensseite:

1. Melden Sie sich bei Ihrem Azure-Konto an, und suchen Sie nach „Azure Defender für IoT“.

1. Für die **Sensorbereitstellung** ist Folgendes erforderlich: Erwerben Sie vorkonfigurierte Hardware von Arrow, oder kaufen Sie Ihr eigenes Gerät, und installieren Sie Sensorsoftware darauf.
1. Für die **Bereitstellung der lokalen Verwaltungskonsole** müssen Sie Ihr eigenes Gerät erwerben und darauf die Software der lokalen Verwaltungskonsole installieren. Weitere Informationen.




## <a name="network-setup"></a>Netzwerkeinrichtung
Nach der Installation der Systemkomponenten von Defender für IoT können Sie Ihre Netzwerkkomponenten für eine optimale Netzwerküberwachung konfigurieren. Weitere Informationen finden Sie im [Leitfaden zur Netzwerkeinrichtung](https://aka.ms/AzureDefenderForIoTNetworkSetup).
