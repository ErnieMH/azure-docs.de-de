# <a name="add-new-publishers-to-the-commercial-marketplace-program"></a>Hinzufügen neuer Herausgeber zum Programm „Kommerzieller Marketplace“

**Geeignete Rollen**

- Besitzer
- Manager

Eine Organisation kann über mehrere Herausgeber verfügen, die einem Konto für den kommerziellen Marketplace zugeordnet sind. Ein Benutzer, der Teil eines vorhandenen Herausgeberkontos ist, kann Herausgeber hinzufügen.

>[!NOTE]
>Überprüfen Sie vor dem Hinzufügen eines neuen Herausgebers die Liste der vorhandenen Herausgeber, indem Sie sich bei Partner Center anmelden und **Einstellungen** > **Kontoeinstellungen** auswählen. Wählen Sie dann im linken Navigationsbereich unter **Organisationsprofil** die Option **Bezeichner** aus. Die Herausgeber in Ihrem Konto sind unter **Herausgeber** aufgeführt.

## <a name="add-new-publishers"></a>Hinzufügen neuer Herausgeber

1. Melden Sie sich bei [Partner Center](https://partner.microsoft.com/dashboard/home) an.
1. Wählen Sie oben rechts **Einstellungen** > **Kontoeinstellungen** aus.
1. Wählen Sie unter **Organisationsprofil** die Option **Bezeichner** aus.
1. Wählen Sie im Abschnitt **Herausgeber** die Option **Herausgeber hinzufügen** aus.
1. Wählen Sie die MPN-ID aus, die Sie dem Herausgeber zuordnen möchten.
1. Aktualisieren Sie die **Herausgeberdetails** im Formular.

   - **Name des Herausgebers**: Der Name, der im kommerziellen Marketplace mit dem Angebot angezeigt wird.  
   - **PublisherID**: Ein Bezeichner, der von Partner Center verwendet wird, um den Herausgeber eindeutig zu identifizieren. Der Standardwert für dieses Feld wird einer vorhandenen und eindeutigen Herausgeber-ID im System zugeordnet. Da die Herausgeber-ID nicht wiederverwendet werden kann, muss dieses Feld aktualisiert werden.  
   - **Kontaktinformationen**: Aktualisieren Sie ggf. die Kontaktinformationen.

Nachdem Sie einen Herausgeber hinzugefügt haben, können Sie mithilfe der Kontoauswahl im linken Navigationsbereich zu diesem Konto wechseln, um im Kontext dieses Kontos zu arbeiten. Sobald Sie zu diesem Herausgeberkonto gewechselt sind, können Sie Ihren Benutzern entsprechende [Benutzerrollen und Berechtigungen](user-roles.md) zuweisen, damit sie Zugriff auf das Programm für den kommerziellen Marketplace für dieses Herausgeberkonto haben.
