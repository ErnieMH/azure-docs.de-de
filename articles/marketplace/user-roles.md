# <a name="assign-user-roles-and-permissions"></a>Zuweisen von Benutzerrollen und Berechtigungen

**Geeignete Rollen**

- Besitzer
- Manager

Um auf Funktionen im Zusammenhang mit dem Marketplace oder Ihrem Entwicklerkonto zugreifen zu können, müssen Ihnen eine oder mehrere der folgenden Rollen zugewiesen sein. Jede Rolle verfügt über einen bestimmten Satz von Berechtigungen, um bestimmte Funktionen innerhalb des Kontos auszuführen.

| Rolle | BESCHREIBUNG |
| ------------ | ------------- |
| Besitzer | Beim Erstellen eines Herausgeberkontos wird dem Partner eine Besitzrolle zugewiesen.<br><br>Berechtigungsdetails:<br><br>&#10004;&#160;&#160;Vollständiger Besitz des Kontos<br>&#10004;&#160;&#160;Hinzufügen eines Benutzertyps<br>&#10004;&#160;&#160;Vervollständigen von Auszahlungs- und Steuerinformationen<br>&#10004;&#160;&#160;Anzeigen von Feedbackberichten und Zusammenfassungen<br>&#10004;&#160;&#160;Registrieren des Herausgeberkontos in anderen Entwicklerprogrammen |
| Manager | &#10004;&#160;&#160;Verwalten von Benutzern, Rollen und Berechtigungen für das Herausgeberkonto<br>&#10004;&#160;&#160;Verwalten von Mandanten<br>&#x2718;&#160;&#160;Zugreifen auf Steuer- und Auszahlungseinstellungen |
| Entwickler | &#10004;&#160;&#160;Hochladen von Paketen<br>&#10004;&#160;&#160;Übermitteln von Apps und Add-Ons<br>&#10004;&#160;&#160;Anzeigen von Nutzungsberichten für Telemetriedetails<br>&#x2718;&#160;&#160;Zugreifen auf finanzielle Informationen oder Kontoeinstellungen |
| Mitwirkender im Geschäftsbereich | &#10004;&#160;&#160;Zugreifen auf finanzielle Informationen<br>&#10004;&#160;&#160;Festlegen von Preisdetails<br>&#x2718;&#160;&#160;Erstellen oder Übermitteln neuer Apps und Add-Ons |
| Mitwirkender im Finanzbereich | &#10004;&#160;&#160;Anzeigen von Auszahlungsberichten<br>&#10004;&#160;&#160;Verwalten von Auszahlungs- und Steuerprofilen<br>&#x2718;&#160;&#160;Durchführen von Änderungen an Apps oder Einstellungen |
| Vermarkter | &#10004;&#160;&#160;Reagieren auf Kundenbewertungen<br>&#10004;&#160;&#160;Anzeigen nicht finanzieller Berichte<br>&#x2718;&#160;&#160;Durchführen von Änderungen an Apps oder Einstellungen |
|||

> [!NOTE]
> Für das Programm „Kommerzieller Marketplace“ werden die Rollen „Globaler Administrator“, „Mitwirkender im Geschäftsbereich“, „Mitwirkender im Finanzbereich“ und „Händler“ nicht verwendet. Das Zuweisen dieser Rollen zu Benutzern besitzt keine Auswirkungen. Nur die Rollen „Manager“ und „Developer“ erteilen Benutzern Berechtigungen.

Weitere Informationen zum Verwalten von Rollen und Berechtigungen in anderen Bereichen von Partner Center, zum Beispiel Azure Active Directory (AD), Cloud Solution Provider (CSP), Control Panel Vendor (CPV), Gastbenutzer oder Microsoft Partner Network (MPN) finden Sie unter [Zuweisen von Rollen und Berechtigungen in Partner Center](/partner-center/permissions-overview).

> [!NOTE]
> Alle Aktivitäten der Benutzerverwaltung und Rollenzuweisung, die in diesen Zeilen durchgeführt werden, stehen im Kontext des Kontos, in dem Sie sich gerade befinden. Lesen Sie den Abschnitt zum Wechseln zwischen Verkäufern, wenn Sie ein anderes Konto verwalten müssen.
