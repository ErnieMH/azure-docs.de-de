# <a name="connect-citrix-analytics-security"></a>Herstellen einer Verbindung mit Citrix Analytics (Security) 

Die Integration von Citrix Analytics (Security) mit Azure Sentinel unterstützt Sie beim Exportieren von Daten, die auf riskante Ereignisse analysiert wurden, aus Citrix Analytics (Security) in Azure Sentinel. Sie können benutzerdefinierte Dashboards erstellen, Daten aus anderen Quellen zusammen mit denen von Citrix Analytics (Security) analysieren und benutzerdefinierte Workflows mit Logic Apps erstellen, um Sicherheitsereignisse zu überwachen und zu minimieren. 



## <a name="prerequisites"></a>Voraussetzungen

- Lese- und Schreibberechtigungen für den Azure Sentinel-Arbeitsbereich.

- Berechtigung für Citrix Security Analytics in Citrix Cloud.


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>Verbinden von Azure Sentinel mit Citrix Analytics (Security)

1. Navigieren Sie im Azure-Portal zu **Azure Sentinel** > **Datenconnectors**, und wählen Sie dann den Connector **Citrix Analytics (Security)** aus.

2. Wählen Sie **Connectorseite öffnen** aus.

3. Befolgen Sie die Anweisungen auf der Seite **Citrix Analytics (Security)** .
