> [!NOTE]
> Bevor Sie mit diesem Tutorial beginnen, sollten Sie das Tutorial [Verbinden des Raspberry Pi-Onlinesimulators mit Azure IoT Hub (Node.js)](../articles/iot-hub/iot-hub-raspberry-pi-web-simulator-get-started.md) oder eines der gerätespezifischen Tutorials abschließen. Sie können beispielsweise zu [Verbinden von Raspberry Pi mit Azure IoT Hub (Node.js)](../articles/iot-hub/iot-hub-raspberry-pi-kit-node-get-started.md) oder zu einer der Schnellstartanleitungen zum [Senden von Telemetriedaten](../articles/iot-hub/quickstart-send-telemetry-dotnet.md) wechseln. In diesen Artikeln richten Sie Ihr Azure IoT-Gerät und IoT Hub ein und stellen eine Beispielanwendung zur Ausführung auf Ihrem Gerät bereit. Die Anwendung sendet gesammelte Sensordaten an Ihren IoT Hub.
