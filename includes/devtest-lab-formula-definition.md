Eine Formel in Azure DevTest Labs ist eine Liste von Standardeigenschaftswerten, die zum Erstellen eines virtuellen Computers (VM, Virtual Machine) verwendet werden. Beim Erstellen eines virtuellen Computers aus einer Formel können die Standardwerte wie vorhanden verwendet oder geändert werden. Ähnlich wie [benutzerdefinierte Images](../articles/devtest-labs/devtest-lab-create-template.md) und [Marketplace-Images](../articles/devtest-labs/devtest-lab-configure-marketplace-images.md) bieten Formeln einen Mechanismus zur schnellen Bereitstellung virtueller Computer. Ähnlich wie [benutzerdefinierte Images](../articles/devtest-labs/devtest-lab-create-template.md) ermöglichen Ihnen Formeln, aus einer VHD-Datei ein Basisimage zu erstellen. Dieses Basisimage kann anschließend zum Bereitstellen eines neuen virtuellen Computers verwendet werden. Um leichter entscheiden zu können, welche Methode sich für Ihre Umgebung besser eignet, lesen Sie die Informationen unter [Vergleich zwischen benutzerdefinierten Images und Formeln in DevTest Labs](../articles/devtest-labs/devtest-lab-comparing-vm-base-image-types.md).
