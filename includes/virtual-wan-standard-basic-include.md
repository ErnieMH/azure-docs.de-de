|---|---|---|
|Basic | Basic | Nur Site-to-Site-VPN |
| Standard | Standard | ExpressRoute<br>Benutzer-VPN (P2S)<br>VPN (Site-to-Site)<br> Übertragung zwischen Hubs und VNET-zu-VNET-Übertragung über den virtuellen Hub |

>Sie können ein Upgrade von Basic auf Standard durchführen, aber nicht von Standard zu Basic zurückkehren.
