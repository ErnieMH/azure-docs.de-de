Für die Arbeit mit Azure Digital Twins in diesem Artikel müssen Sie zuerst *eine Azure Digital Twins-Instanz einrichten*. Außerdem benötigen Sie die erforderlichen Berechtigungen für ihre Verwendung. Wenn Sie bereits eine Azure Digital Twins-Instanz eingerichtet haben, können Sie diese stattdessen nutzen.

Befolgen Sie andernfalls die Anleitung unter [Einrichten einer Instanz und der Authentifizierung](../articles/digital-twins/how-to-set-up-instance-portal.md). Die Anweisungen enthalten Informationen, mit denen Sie überprüfen können, ob jeder Schritt erfolgreich abgeschlossen wurde.

Notieren Sie sich nach dem Einrichten der Instanz die folgenden Werte. Sie benötigen diese Werte später, um eine Verbindung mit der Instanz herzustellen:
* *Hostname* der Instanz. Sie [finden den Hostnamen im Azure-Portal](../articles/digital-twins/how-to-set-up-instance-portal.md#verify-success-and-collect-important-values).
* Das *Azure-Abonnement*, das Sie zum Erstellen der Instanz verwendet haben. Sie können den Namen oder die ID verwenden. Sie finden das Abonnement im [Azure-Portal](https://portal.azure.com) auf der Seite **Übersicht** Ihrer Instanz.
