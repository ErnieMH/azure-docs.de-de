| Resource | Begrenzung |
| --- | --- |

| Storage-Kontoverwaltungsvorgänge (Schreiben) |10 pro Sekunde/1.200 pro Stunde |
| Storage-Kontoverwaltungsvorgänge (Liste) |100 pro 5 Minuten |
