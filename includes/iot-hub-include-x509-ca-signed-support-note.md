> [!IMPORTANT]
> Die folgenden Funktionen werden bei Geräten, die die Authentifizierung per X.509-Zertifizierungsstelle (CA) verwenden, nicht unterstützt:
>
> - HTTPS, MQTT über WebSockets und AMQP über WebSockets-Protokolle.
> - Dateiuploads (alle Protokolle).
>
> Dieses Feature wird auf Geräten unterstützt, die die X.509-Fingerabdruckauthentifizierung verwenden. Weitere Informationen zur X.509-Authentifizierung mit IoT Hub finden Sie unter [Unterstützte X.509-Zertifikate](../articles/iot-hub/iot-hub-devguide-security.md#supported-x509-certificates).  
>
