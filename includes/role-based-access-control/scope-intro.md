Wenn Sie Rollen zuweisen, müssen Sie einen Bereich angeben. Ein Bereich ist der für den Zugriff geltende Ressourcensatz. In Azure können Sie einen Bereich auf vier Ebenen angeben (von weit nach eng): [Verwaltungsgruppe](../../articles/governance/management-groups/overview.md), Abonnement, [Ressourcengruppe](../../articles/azure-resource-manager/resource-group-overview.md#resource-groups) und Ressource.
