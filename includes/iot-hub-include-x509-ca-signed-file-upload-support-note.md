> [!IMPORTANT]
> Die Funktion zum Hochladen von Dateien auf Geräten, die die Authentifizierung der X.509-Zertifizierungsstelle verwenden, befindet sich in der öffentlichen Vorschau, und der [Vorschaumodus muss aktiviert werden](../articles/iot-hub/iot-hub-preview-mode.md). Dieses Feature ist auf Geräten, die die X.509-Fingerabdruckauthentifizierung verwenden, allgemein verfügbar. Weitere Informationen zur X.509-Authentifizierung mit IoT Hub finden Sie unter [Unterstützte X.509-Zertifikate](../articles/iot-hub/iot-hub-dev-guide-sas.md#supported-x509-certificates).
