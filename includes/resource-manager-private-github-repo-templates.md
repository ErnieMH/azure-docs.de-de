> [!NOTE]
> Um eine Vorlage bereitzustellen oder auf eine verknüpfte Vorlage zu verweisen, die in einem privaten GitHub-Repository gespeichert ist, finden Sie eine benutzerdefinierte Lösung, die in einem [MVP-Blog](https://blog.tyang.org/2017/05/19/deploying-arm-templates-with-artifacts-located-in-a-private-github-repository/) dokumentiert ist. Sie können eine [Azure-Funktion](../articles/azure-functions/functions-overview.md) als Proxy einrichten, um die URL zu erstellen, die für den Zugriff auf eine Vorlagendatei in einem privaten GitHub-Repository erforderlich ist.
