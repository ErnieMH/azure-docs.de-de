Sie können die Azure PowerShell-Cmdlets auch lokal auf Ihrem Computer installieren und ausführen. PowerShell-Cmdlets werden regelmäßig aktualisiert. Wenn Sie nicht die aktuelle Version installiert haben, kann mit den in diesen Anweisungen verwendeten Werten ein Fehler auftreten. Verwenden Sie das Cmdlet `Get-Module -ListAvailable Az`, um die auf Ihrem Computer installierten Azure PowerShell-Versionen zu ermitteln. Informationen zum Installieren oder Aktualisieren finden Sie unter [Installieren des Azure PowerShell-Moduls](/powershell/azure/install-az-ps).
