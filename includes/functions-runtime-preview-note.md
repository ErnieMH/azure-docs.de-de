> [!IMPORTANT]
> Die Azure Functions-Runtime, Preview 2, unterstützt nur die Version 1.x der Azure Functions-Runtime. Diese Previewfunktion wird nicht aktualisiert, um Version 2.x oder höher der Runtime zu unterstützen, und es sind auch keine zukünftigen Updates geplant. Wenn Sie die Azure Functions-Runtime außerhalb von Azure hosten müssen, sollten Sie [die Verwendung von Azure Functions in Kubernetes mit KEDA](../articles/azure-functions/functions-kubernetes-keda.md) erwägen.
