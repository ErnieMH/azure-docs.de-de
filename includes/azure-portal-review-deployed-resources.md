Vergewissern Sie sich, dass Sie im Dashboard des Azure-Portals Daten zum virtuellen Computer anzeigen können.

1. Wählen Sie im Azure-Portal **Dashboard** aus.

    :::image type="content" source="media/azure-portal-review-deployed-resources/navigate-to-dashboards.png" alt-text="Azure-Portal-Navigation zum Dashboard":::

1. Wählen Sie auf der Seite „Dashboard“ **Einfaches VM-Dashboard** aus.

    :::image type="content" source="media/azure-portal-review-deployed-resources/select-simple-vm-dashboard.png" alt-text="Navigieren zu einfachem VM-Dashboard":::

1. Überprüfen Sie das Dashboard, das von der Azure Resource Manager-Vorlage (ARM-Vorlage) erstellt wurde. Sie können sehen, dass ein Teil des Inhalts statisch ist, aber es gibt auch Diagramme, die die Leistung des virtuellen Computers anzeigen, den Sie am Anfang erstellt haben.

    :::image type="content" source="media/azure-portal-review-deployed-resources/review-simple-vm-dashboard.png" alt-text="Überprüfen des einfachen VM-Dashboards":::
